Langage web: Java

Java est un langage de programmation orienté objet. Ce langage a été créé en 1995 par deux employés d'une entreprise de fabrication d'ordinateur.

Le Java fut créé afin d'améliorer les performances du langage C et C++. Mais finalement, il fut adopté comme langage à part entière. Du coup, il finira par s'intégrer. au langage du Web dans des navigateurs avant de finir dans des jeux vidéo sur Internet avant de se faire remplacer par Adobe Flash Player.
